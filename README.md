## 📋 Sobre

Este é meu site de portfólio profissional, onde compartilho:

- 💼 **Experiências profissionais** em Engenharia de Dados
- 🚀 **Projetos pessoais** e técnicos desenvolvidos
- 🎓 **Certificações** e formações acadêmicas
- 🛠️ **Habilidades técnicas** em Python, SQL, Cloud e Big Data

## 🛠️ Tecnologias

- **React 18**: Componentes e interface
- **Tailwind CSS**: Estilização e design responsivo
- **GitHub Pages**: Hospedagem e deploy automático

## 📂 Estrutura

```
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos customizados
└── js/
    ├── app.js          # Aplicação principal
    ├── data/           # Dados dos projetos, experiências, etc.
    ├── components/     # Componentes reutilizáveis
    ├── sections/       # Seções da página
    └── pages/          # Páginas da aplicação
```

## 🚀 Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/isabellazramos/isabellazramos.github.io.git
cd isabellazramos.github.io

# Inicie um servidor local
python -m http.server 8000

# Acesse: http://localhost:8000
```

## 📝 Atualizar Conteúdo

O conteúdo do portfólio pode ser atualizado editando os arquivos na pasta `js/data/`:

- `projects.js` - Projetos
- `experiences.js` - Experiências profissionais
- `certifications.js` - Certificações
- `skills.js` - Habilidades técnicas
